Argos

HTML5 Mobile Solution

Download .zip View on GitHub

Take Mobile HTML5 Apps to the next level with Argos - a full featured mobile application framework that enables seamless integration with external data and handles all the under the cover details letting you focus on the end application.

Compatibility

Works in all HTML5 compatible browsers both Mobile and Desktop

History

Navigation history and context system that drives the state and URL; enabling users to pick up where they left off

AMD

Every file is a module and loaded asynchronously to provide clear code structure and faster load time

Dojo

Built upon the base Dojo classes, any piece of Dojo may be optionally pulled in and used

Dijit

All View modules are Dijits meaning all the standard dojo dijit methods and properties apply

Simplate

Every bit a markup is driven using the powerful-but-simple Simplate templating engine

Transitions

Optimized transitions for clear depth navigation and a fluid user experience
